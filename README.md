# チーム塩大福

## 概要
チーム塩大福は、 Minecraft を中心に活動するクリエイティブチームです。リーダーである塩雪（@sioyuki20213）を中心に、アドベンチャーワールドの制作や建築イベントへの参加、観光配信を行っています。オリジナルアドオンやコマンドを活用したテーマパーク風のワールド構築を得意とし、メンバー同士のコラボレーションで楽しく創作を進めています。

## メンバー
- **リーダー**: 塩雪 (@sioyuki20213) - Minecraft コマンド/アドオン勉強中。 YouTube チャンネル「しおゆきのしっぽ」でライブ配信活動中。
- **メンバー**: 現在約20名。建築勢だけでなく様々な個性豊かなメンバーを中心に、観光やミニゲーム制作で協力。
